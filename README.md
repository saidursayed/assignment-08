# App Name
HERO.IO

## Description
A modern, responsive, and fully functional App Store–style web application built for Assignment 008.  
Hero IO allows users to explore apps, view details, check ratings, install/uninstall apps, and manage installations — all with a clean UI and smooth UX.


## Technologies Used
- React.js  
- React Router  
- Tailwind CSS  
- DaisyUI  
- Axios  
- Recharts  
- Vite  
- React Icons
- Google Font
